Cheat Sheet

DOCTYPE: The Doctype declaration defines the document type and version of HTML being used, ensuring the browser renders the page correctly.

body: The body element contains the content of the HTML document, such as text, images, links, tables, etc., that will be visible on the web page.

em: The em element is used to emphasize text, usually rendering it in italics by default.

head: The head element contains metadata about the HTML document, such as links to stylesheets, scripts, the title, and meta information.

html: The html element is the root element of the HTML document, containing all other HTML elements.

strong: The strong element indicates that the enclosed text is of strong importance, usually rendering it in bold.

title: The title element specifies the document's title, which appears in the browser's title bar or tab.
